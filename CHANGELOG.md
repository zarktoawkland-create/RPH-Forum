## [2026-04-06] 角色卡违禁词审核系统

### 新增功能

#### 1. 角色卡上传违禁词检测

上传角色卡时，系统自动解析角色卡内容（支持 PNG 和 JSON 格式），对以下字段进行违禁词扫描：

- 角色名称、描述、性格设定、场景设定
- 开场白、对话示例、创作者备注
- 系统提示词、对话历史指令
- 备用开场白、标签
- 世界书（Character Book）条目内容

若检测到违禁内容，上传将被拒绝，返回具体违规字段和匹配信息，并记录操作日志。

#### 2. 管理后台 - 违禁词检测页面

在管理后台导航栏新增「违禁词检测」模块（盾牌图标），提供以下功能：

- **审核开关**：全局开启/关闭违禁词检测
- **严格模式**：开启后命中任何违禁词即拒绝（可扩展用途）
- **违禁词列表**：查看所有自定义违禁词规则，支持启用/禁用、编辑、删除
- **添加违禁词**：通过弹窗添加新的违禁词规则
  - 正则表达式输入，支持实时语法验证
  - 类型选择：自定义 / XSS 防护 / 注入防护 / URL 防护 / 内容过滤
  - 严重程度选择：低 / 中 / 高 / 严重
  - 可选描述信息

#### 3. 内置安全规则

预置以下安全检测规则（不可删除）：

| 规则 | 说明 | 严重程度 |
|------|------|----------|
| XSS 脚本注入 | `<script>` 标签检测 | 高 |
| JavaScript 协议 | `javascript:` 协议检测 | 高 |
| 事件处理器注入 | `onclick=` 等 DOM 事件 | 高 |
| iframe / object / embed 注入 | 外部嵌入标签检测 | 高 |
| 数据 URL 协议 | `data:text/html` 等 | 中 |
| 表达式注入 | `${}` 模板表达式 | 中 |

### 新增 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/admin/audit-patterns` | 获取违禁词列表 |
| POST | `/api/admin/audit-patterns` | 添加违禁词 |
| PUT | `/api/admin/audit-patterns/:id` | 编辑违禁词 |
| DELETE | `/api/admin/audit-patterns/:id` | 删除违禁词 |

### Bug 修复

- 修复 `PUT /api/admin/settings` 路由中非标签设置（如 `audit_enabled`、`audit_strict_mode`）无法保存的问题。原逻辑对所有非标签设置执行 `continue` 跳过，现已修正为仅对标签设置做校验，其余设置正常保存。

### 使用说明

1. 登录管理后台 `/admin`
2. 导航栏点击「违禁词检测」
3. 开启审核开关
4. 点击「添加违禁词」配置自定义规则
5. 上传角色卡时系统自动检测
